První projekt Engeto akademie: Task manager 
Zadání č. 1
Vaším úkolem je doplnit prázdné funkce pro správu úkolů ve správném pořadí. Program by měl umožnit přidávat, zobrazovat a odstraňovat úkoly. Následující části kódu jsou prázdné funkce, které musíte doplnit. 
Každá funkce má svůj specifický úkol, který je popsán níže. Úkoly budou ukládány do seznamu ukoly = [].

def hlavni_menu()
Funkce hlavního menu, která poskytuje možnosti pro přidání, zobrazení a odstranění úkolu. Pokud uživatel zadá neplatnou volbu, program ho upozorní a nechá uživatele opakovat znovu volbu.

def pridat_ukol()
Tato funkce má uživateli umožnit zadat název a popis nového úkolu a uložit jej do seznamu úkolů. Zde platí volba 1 v hlavním menu. Po zadání úkolu program pokračuje dál nabídkou hlavního menu. 
Při zadání prázdného vstupu do Zadejte název úkolu nebo Popis úkolu, program upozorní uživatele, že zadal prázdný vstup a nechá ho zadat název i popis znovu.

def zobrazit_ukoly()
Tato funkce má zobrazit všechny úkoly v seznamu. Zde platí volba 2 v hlavním menu. Po zobrazení úkolů program pokračuje dál nabídkou hlavního menu.

def odstranit_ukol()
Tato funkce má uživateli umožnit zadat číslo úkolu, který chce odstranit, a tento úkol odstranit. Zde platí volba 3 v hlavním menu. Po odstranění úkolu program pokračuje dál nabídkou hlavního menu. 
Zde je potřeba, aby uživatel viděl všechny uložené úkoly a při výběru neexistujícího úkolu byl upozorněn.

Konec programu
Pokud uživatel zadá volbu 4 v hlavním menu program se ukončí.
